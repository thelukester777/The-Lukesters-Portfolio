'data-link' Attribute

This is actually the main function for this website. You can see it's very simple and just swaps the content based on the ID of each section. It does this by finding a matching 'data-link' attribute on the anchor tags. Rather than have many indivdual pages to link to and keep track of, I kept everything in one HTML document. It might not be great for SEO, but it makes it easy to edit/interact with. Very minimal pagenation. Very simple jQuery function.


var a = $('a'), // Assign a variable to DOM Selectors.
    cN = '', // Hoist the variable to make more global.
    mN = $('.main'),
    nV = $('nav, header');

$(a).click(function() {
    var cN = $(this).attr('data-link'); // Assigns 'data-link' attribute3 as class.
    $(mN).hide(); // Hides the default class so that it can be replaced.
    $('#' + cN).fadeIn('slow'); // Uses new attribute and creats ID selector.
    $(nV).removeClass().addClass(cN); // Swaps navs for each page with selector.
});

Check Query String

This is also a snippet from this website. I use this function to retrieve the query string params. I use them for various admin purposes as well as defining which content should show once user navigates from another page, like the "Thank You" page.


// Query String Fn
var pU = '',
    pP = '',
    pN = '';

function getUrlParams(q) { // Fetches the value of a URL query param.
var pU = window.location.search.substring(1),
    pP = pU.split('&');
    for (var i = 0; i < pP.length; i++) { //Loop through URL query string
        var pN = pP[i].split('=');
        if (pN[0] == q) { // Checks the param name(s)
            return pN[1]; // Returns the param value
        }
    }
}
// Initiates the function.
getUrlParams();

// Alternate way to initiate and return the value 'design' from '?page=design'
var page = getUrlParams('page');

Ajax Request

Working on an ecommerce website, you work with many API's. Whether you are looking at services for profile information, shopping cart, product information, or working with 3rd party data, it helps to know how to utilize and manipulate that information. Using Ajax is pretty standard.


$.ajax({
    url: 'https://thelukester777.github.io/portfolio/some/fake/server/database/get', // Not Real :)
    dataType: 'json', // We want the data to come in JSON format.
    success: function(data) { // If the request goes through successfully, continue.
        //Takes data, assigns variable to container and parses
        var c1 = $('.container-one'), // Assigns container.
            c2 = $('.container-two');
        $(c1).append(data.valueOne); // Appends value 'valueOne' to '.container-one'
        $(c2).append(data.valueTwo);
        console.log('Dev note: Data loaded successfully!'); // Console log to show successful data load.
    },
    error: function(xhr, ajaxOptions, thrownError) { // Error handling.
        if (xhr.status === 404) { // If the API 404 errors, you may be able to resolve. So log it.
                    console.log('Dev note: ' + xhr.status + ' ' + xhr.statusText + '. data not loaded.');
        } else if (xhr.status !== 404) { // If it's not 404ing, it might be something else with service. 
                    console.log('Dev note: ' + xhr.status + ' ' + xhr.statusText + '. Check API status.');
        }
    }
});

"Faces" - 2009

'Faces' is about not smiling. It represents the silly twisted faces people make when they aren't in the mood to pose for the camera. People in these moments typically reveal something you just don't get by telling them to "Say cheese!" I used the opportunity to play with light reflection and hunted for color combinations that compliment the mood of their expressions. While trying to keep them somewhat photo-realisitic—I amped up colors and warped features slightly to get a more animated look.

"The Alley Stores" - 2006-2007

While in college, I worked at The Alley stores, a popular punk-rock/ metal haven for buying alternative clothing and accessories. Over the course of a year, I designed over 50 t-shirt logos and even helped print a great deal of them. It was a very fun job and it was the first time I considered myself a commercial artist.

"Dare to Dream" - 2013

I got the idea for this after seeing a news bit about a turtle that had been tied to the end of a weather balloon. I was inspired to make this cruel story more light hearted. I decided that instead to portray a turtle who has always dreamed of flying, and now has reached his dream.

"Octopi Your Mind" - 2014

I was inspired by seeing a few drawings of octopi, and I thought it would be really fun to try creating my own version of one. I kept layering textures and playing with various colors until I got the array just right. It lacks a bit in technicality, but the colors are what makes this one special to me.

"Sea Floor Fight" - 2014

This was a fun idea I had shortly after re-touching the "Optopi Your Mind" piece. I was challeneged by a friend to start drawing again, so I took my sketchbook with me on my commutes for a week and came up with the first draft of this in pencil.
